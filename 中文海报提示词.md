# 要求
## 尺寸与基础结构
- 比例：严格为 4:3 (宽高比）
- 边框设计：外边缘为 div 布局，生成图时无明显边界
- 外圈卡片需为直角，干净利落
- 使用 3–4 种不同字号强调层次，关键词字号为最大字级；关键词提取 2–3 个关键词，可加描边、高亮、或不同颜色处理；



## 设计要求
- 绝对禁止使用紫色，渐变色；
- 绝对禁止使用AI味浓厚的图标；
- 运用反差空间与比例技巧制造视觉焦点
- 文字排版：保持阅读节奏与比例协调
- 动效设计：可用轻微渐变、阴影或模糊制造层次感
- 突出核心卖点，引导读者视线移动 

## 技术实现要求
- 添加一个不影响设计的保存按钮，使用html2canvas实现一键保存成图片功能；
- 使用 Google Fonts 或 CDN 字体资源（如思源黑体、Inter）
- 可用 Font Awesome 添加图标美化
