## 2. 生成参数

- **关键词提取**：
  1. **自媒体**（主关键词，最大字号 180px）

## 3. 整体风格总结

### 设计灵感来源
本设计借鉴了三位经典设计大师的风格精髓：

1. **Josef Müller-Brockmann（约瑟夫·米勒-布罗克曼）**
   - 瑞士国际主义风格代表人物
   - 借鉴元素：精确的网格系统、数学化的布局结构
   - 转译方式：背景使用 40px 标准网格，所有元素基于网格对齐

2. **Piet Mondrian（皮特·蒙德里安）**
   - 荷兰风格派运动创始人
   - 借鉴元素：几何抽象、原色色块、黑色线条分割
   - 转译方式：使用红色和橙色几何色块作为视觉锚点，黑色线条形成边界装饰

3. **Paula Scher（葆拉·雪儿）**
   - 当代平面设计大师，字体排版专家
   - 借鉴元素：超大字号、大胆的字体层次对比、高冲击力排版
   - 转译方式：180px 主标题，96px 副标题，形成强烈的视觉节奏

### 视觉层次
- **第一视觉层**：超大字号"自媒体"（180px，黑色）
- **第二视觉层**："媒体"关键词高亮（红色 + 橙色底部高光）
- **第三视觉层**："运营实践"副标题（96px，黑色）
- **第四视觉层**：几何色块装饰（红色、橙色、黑色线条）
- **第五视觉层**：描述性文字（32-38px，灰色为主，关键词红色）

### 色彩心理学应用
- **黑色**：专业、权威、现代感
- **红色 (#FF4136)**：热情、行动力、吸引注意力
- **橙色 (#FF851B)**：活力、创造力、温暖感
- **白色**：简洁、纯净、呼吸空间

### 排版特点
- **字号对比**：4 种字号层次（180px / 96px / 38px / 32px），比例约为 5.6:3:1.2:1
- **字重对比**：900（超粗）/ 700（粗体）/ 600（半粗）/ 400（常规）
- **字间距调整**：主标题 -0.02em（紧凑），副标题 0.05em（舒展），描述文字 0.1em（松散）
- **行高控制**：主标题 0.95（紧密），描述文字 1.8（舒适阅读）

## 4. 生成思路

### 设计理念
基于"内容为王"的自媒体核心理念，采用极简主义与几何抽象相结合的方式，通过强烈的字体对比和几何色块，营造出专业、现代、高效的视觉印象。

### 创作流程

#### 第一步：确定视觉风格
- 分析需求：抖音平台特性（年轻化、视觉冲击力强）
- 选择灵感来源：网格系统（理性）+ 几何色块（活力）+ 大胆字体（冲击力）
- 避开设计禁忌：不使用紫色、渐变色、AI 感图标

#### 第二步：构建布局结构
- 采用 40px 基础网格系统
- 主要内容居中垂直排列
- 几何色块分布于四周边缘，不干扰主要信息
- 预留足够的呼吸空间（padding: 80px 100px）

#### 第三步：字体层次设计
- 提取关键词："自媒体"、"运营"、"实践"
- 将"自媒体"作为主视觉焦点，使用 180px 超大字号
- "媒体"二字用红色高亮，底部橙色色块强化
- "运营实践"作为副标题，96px 字号形成对比
- 描述性文字使用小字号（32-38px），提供辅助信息

#### 第四步：色彩与装饰
- 主色调：黑白为主（占 70%）
- 强调色：红色和橙色（占 30%）
- 几何色块布局：
  - 左上红色竖条（高度 280px）- 引导视线向下
  - 右上黑色横条（宽度 180px）- 平衡构图
  - 左下橙色横条（宽度 320px）- 稳定画面
  - 右下黑色竖条（高度 220px）- 形成闭合
  - 两个半透明方块（红色、橙色）- 增加层次

#### 第五步：交互功能实现
- 使用 html2canvas 库实现一键保存
- 保存分辨率设置为 2x（高清输出）
- 添加简单的页面加载动画（淡入 + 上移）
- 保存按钮独立于海报主体，不影响设计美观

### 技术实现要点

#### HTML 结构
```
海报容器 (1200×900px)
├─ 背景网格层
├─ 几何色块层（6个装饰色块）
├─ 装饰文字层
└─ 主内容层
   ├─ 标题区（主标题 + 副标题）
   ├─ 分隔线
   └─ 描述文字
```

#### CSS 关键技术
- Flexbox 居中布局
- 绝对定位实现几何色块
- 伪元素 `::after` 实现关键词底部高光
- 多层次阴影增强视觉深度
- 响应式断点：1400px / 1024px / 768px

#### JavaScript 功能
- `savePoster()` 函数：调用 html2canvas，生成 PNG 图片
- `window.onload` 事件：实现入场动画效果
- 错误处理：捕获保存失败情况并提示用户

### 设计特色

1. **高辨识度**：超大字号 + 关键词高亮，3 秒内传达核心信息
2. **视觉平衡**：中心对称的主体内容 + 四周不对称的装饰色块
3. **节奏感强**：通过字号、字重、颜色的多层次对比，形成视觉韵律
4. **易于传播**：符合抖音平台特性，清晰的主题表达，适合移动端浏览
5. **可复用性**：更换文案即可快速生成同系列封面

- ✅ 小红书笔记封面
- ✅ B站视频封面

## 5. 使用说明

### 本地预览
1. 直接打开 `index.html` 文件
2. 在浏览器中查看效果（推荐使用 Chrome）

### 保存图片
1. 点击页面下方的"保存为图片"按钮
2. 图片将自动下载到本地
3. 文件名格式：`自媒体运营实践_抖音封面_时间戳.png`

### 修改文案
如需修改文案，编辑 `index.html` 文件中的以下部分：

```html
<!-- 第 154-160 行：主标题区域 -->
<h1 class="main-title">
    自<span class="highlight">媒体</span>
</h1>

<!-- 第 163 行：副标题 -->
<h2 class="subtitle">运营实践</h2>

<!-- 第 171-173 行：描述文字 -->
<p class="description">
    从<span class="description-highlight">策略</span>到<span class="description-highlight">执行</span><br>
    打造高效内容生态
</p>
```

### 调整颜色
主要颜色变量位于 CSS 中，可统一修改：
- 红色：`#FF4136` → 搜索替换为其他颜色
- 橙色：`#FF851B` → 搜索替换为其他颜色
- 黑色：`#000000` → 搜索替换为其他颜色
