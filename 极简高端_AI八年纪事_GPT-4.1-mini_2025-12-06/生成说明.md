## 3. 整体风格总结
- 结构上左文右图：左侧以标题与时间线文字为主，构成信息密度与节奏；右侧以“八年时间跨度 + 过去/现在对比卡片”作为视觉焦点。
- 色彩采用深色背景 + 少量绿色高光（#06d6a0），完全避开紫色与渐变主色，仅在局部使用轻微径向渐变制造空间感。
- 字体使用 Google Fonts `Noto Sans SC` 与 `Inter`，配合 3–4 个字号梯度，清晰区分主标题/副标题/列表。
- 通过留白与分栏，保证在B站封面缩略图尺寸下依然可以快速读取“2017-2025 / AI / 8年重塑一切”等核心信息。

## 4. 生成思路与实现要点
1. **围绕3:4比例的画布设计**
   - 使用 CSS `aspect-ratio: 3 / 4` 保证在不同屏幕宽度下仍保持封面比例。
   - 外层 `poster-wrapper` 采用直角卡片，搭配轻微投影，不添加明显的边框线，符合“生成图时无明显边界”的要求。

2. **信息分层与阅读节奏**
   - 顶部：品牌与平台信息条（`AI 编年史` + `B站长视频·深度解读`），用小字号与胶囊型背景弱化存在感但强化专业感。
   - 中部左侧：主标题 + 副标题 + 问题句，核心关键词如“重塑一切”“对话纪元”使用反差色块或下划线突出。
   - 中部右侧：年份跨度 + “过去 vs 现在”对比卡片，帮助用户一眼理解视频的时间维度与主旨。
   - 底部：观看收益与引导文案（收藏提示），满足B站封面兼顾“告知 + 行动引导”的特性。

3. **动效与层次感处理**
   - 未使用明显的动态动画，避免干扰导出图片；采用阴影、径向渐变、半透明描边，模拟“景深”和“光晕”。
   - 标题关键词使用浅色实底 + 深色背景，形成强对比视觉焦点。

4. **技术实现（html2canvas 一键保存）**
   - 在页面中添加一个不影响设计的底部工具栏，按钮文字为“保存封面为图片”。
   - 使用 CDN 引入 `html2canvas@1.4.1`，在 `main.js` 中对 `id="poster-root"` 的区域进行截图。
   - 通过 `scale: 2` 提升导出图片清晰度，并自动按照日期生成文件名（例如 `poster-20251206.png`）。

5. **可扩展方向**
   - 可以在右侧卡片中加入更精细的“年份节点”微型时间线，用细线和小圆点表示八年中的关键年份。
   - 可根据不同平台（如小红书、公众号）复制当前结构，仅调整比例与标题字号，即可快速生成不同版本的封面。
