# 生成说明文档

## 1. 生成工具与模型
- **生成工具**：Claude Code CLI
- **调用模型**：Claude Sonnet 4.5 (claude-sonnet-4-5-20250929)
- **生成日期**：2025-12-06

## 2. 生成参数

### 场景
B站视频封面图

### 尺寸
- 宽度：900px
- 高度：1200px
- 宽高比：3:4（严格遵循）

### 分辨率
- 基础分辨率：900×1200px
- 导出分辨率：1800×2400px（2倍scale）
- 输出格式：PNG

### 风格
极简高端 - 现代商业风

### 平台
B站（Bilibili）视频封面

### 文案
**主题**：2017-2025，AI如何用八年重塑一切

**章节结构**：
- 序章：无声的惊雷 (2017)
- 第一幕：巨兽的蛰伏 (GPT-1/2/3)
- 第二幕：创世的火焰 (2022图像大爆发)
- 第三幕：对话纪元开启 (ChatGPT的"iPhone时刻")
- 第四幕：诸神黄昏与平民英雄 (开源与DeepSeek)
- 第五幕：智能体的觉醒 (Devin & Manus)
- 第六幕：重构现实 (Sora、Kling等多模态)
- 终章：无尽的未来 (Suno/Udio与展望)

## 3. 整体风格总结

### 视觉风格
- **色彩方案**：黑色背景 (#0a0a0a - #1a1a1a) + 白色文字 + 橙色强调色 (#ff6b35)
- **设计理念**：极简主义 + 高端商业感 + 科技未来感
- **字体层级**：4级字号体系（120px / 90px-110px / 36px / 16-18px）
- **几何装饰**：简洁的方形、圆形元素，低透明度，不抢戏

### 布局特点
- **上部**：时间线（2017-2025），建立时间框架
- **中部**：核心主题文案，视觉焦点，"八年"字号最大
- **下部**：8个章节的双列网格排布，内容丰富但不杂乱
- **底部**：橙色强调线，收束视觉

### 设计亮点
1. **关键词突出**："八年"使用超大字号（120px）+ 橙色 + 阴影效果
2. **视觉层次**：通过字号、颜色、字重创造清晰的信息层级
3. **空间利用**：合理的留白与内容密度平衡
4. **可读性强**：使用思源黑体（Noto Sans SC），中文显示优秀

## 4. 生成思路

### 设计策略
1. **避免AI味**：
   - 不使用渐变色、紫色等典型AI配色
   - 不使用科技感光效、粒子等过度AI化元素
   - 采用商业设计的极简风格

2. **符合B站调性**：
   - 清晰的信息传达，适合快速浏览
   - 视觉冲击力强，能在推荐流中脱颖而出
   - 专业感与内容深度相匹配

3. **层次构建**：
   - 第一视觉焦点："八年"（最大字号 + 强调色）
   - 第二视觉焦点："重塑一切"（大字号标题组）
   - 第三层信息：时间跨度（2017-2025）
   - 细节信息：8个章节的内容预告

4. **功能实现**：
   - 使用html2canvas实现高质量截图
   - scale设为2倍，确保导出图片清晰
   - 添加页面加载动画，提升视觉体验

### 技术选型
- **字体**：Google Fonts的Noto Sans SC（思源黑体），中文显示效果佳
- **截图库**：html2canvas v1.4.1，成熟稳定
- **响应式**：提供3档断点适配（960px / 640px / 默认）

### 用户体验
- 一键保存功能，操作简单
- 保存按钮放置在海报外部，不影响设计
- 按钮状态反馈（生成中...），防止重复点击
- 文件名自动添加日期时间戳

## 5. 使用方法

1. 用浏览器打开 `index.html` 文件
2. 预览封面效果
3. 点击"保存为图片"按钮
4. 图片将自动下载到本地
