# 生成说明文档

- **生成工具**：Claude Code CLI
- **调用模型**：Claude Sonnet 4.5 (claude-sonnet-4-5-20250929)
- **生成日期**：2025-12-06

## 2. 生成参数

### 场景
B站视频封面图

### 尺寸
- 宽度：1600px
- 高度：1200px
- 宽高比：4:3（横版，严格遵循）

### 分辨率
- 基础分辨率：1600×1200px
- 导出分辨率：3200×2400px（2倍scale）
- 输出格式：PNG

### 风格
科技简约 - 现代商业风格

### 平台
B站（Bilibili）视频封面

### 文案
**主要文案**：2017-2025，AI如何用八年重塑一切

**次要文案（精选展示）**：
- 第一幕：巨兽蛰伏 (GPT萌芽)
- 第二幕：创世火焰 (图像爆发)
- 第三幕：对话纪元 (ChatGPT时刻)
- 第四幕：开源崛起 (DeepSeek)
- 第五幕：智能体觉醒 (Agent时代)
- 第六幕：多模态 (Sora重构)
- 终章：无尽的未来

## 3. 整体风格总结

### 视觉风格
- **色彩方案**：纯黑背景 (#000000) + 白色文字 (#FFFFFF) + 红色强调 (#dc2626)
- **设计理念**：科技感 + 极简主义 + 力量感
- **字体系统**：
  - 中文：思源黑体 (Noto Sans SC)
  - 英文/数字：Inter（现代无衬线字体）
- **字号层级**：4级体系（140px / 96-110px / 52px / 14-28px）

### 布局特点
- **横版布局**：4:3比例，更适合桌面端观看
- **左右分区**：
  - **左侧**：主标题区，占据视觉焦点
  - **右侧**：章节信息区，结构化内容
- **视觉动线**：从左上（时间）→ 中央（关键词"八年"）→ 右侧（章节）
- **装饰元素**：
  - 背景网格（低透明度，增加科技感）
  - 竖向红色线条（左侧）
  - 横向红色线条（底部）

### 设计亮点
1. **关键词突出**："八年"使用超大字号（140px）+ 红色 + 动态阴影效果
2. **层次分明**：通过字号、颜色、字重创建清晰的信息层级
3. **卡片设计**：右侧6个章节卡片，简洁且信息密度适中
4. **无尽未来**：底部使用无穷符号（∞）+ 独立样式，强调展望

## 4. 生成思路

### 设计策略
1. **避免AI味**：
   - 严格避免紫色、渐变色
   - 不使用光效、粒子等AI常见元素
   - 采用纯色 + 极简几何的专业设计手法

2. **横版优势**：
   - 4:3比例更适合B站播放器预览
   - 左右分栏让信息层次更清晰
   - 更多横向空间展示核心内容

3. **视觉焦点策略**：
   - **第一焦点**："八年"（最大字号 + 红色 + 呼吸光效）
   - **第二焦点**："重塑"（次大字号 + 白色）
   - **第三焦点**：时间跨度（2017-2025）红色标注
   - **辅助信息**：右侧章节卡片网格

4. **信息密度控制**：
   - 主文案完整呈现
   - 次要文案精选6个章节 + 1个终章
   - 避免信息过载，保持视觉呼吸感

5. **互动体验**：
   - 页面加载时的渐入动画
   - 鼠标悬停章节卡片的微交互
   - 关键词的呼吸光效（2秒周期）

### 技术实现
- **字体加载**：同时加载思源黑体和Inter，中英文混排效果佳
- **截图优化**：
  - scale设为2倍，确保3200×2400高清输出
  - 明确指定背景色避免透明度问题
- **动画系统**：
  - CSS transition + JavaScript 控制
  - 分层级的时序动画（标题→卡片→标签）
  - 关键词呼吸效果使用setInterval实现

### 响应式设计
提供3档断点适配：
- 1680px以下：缩小至1200×900
- 1280px以下：缩小至960×720
- 保持4:3比例，等比缩放字号和间距

## 5. 使用方法

1. 用浏览器打开 `index.html` 文件
2. 等待字体加载和动画播放
3. 预览封面效果
4. 点击"保存为图片"按钮
5. 图片将自动下载到本地（文件名包含日期）
