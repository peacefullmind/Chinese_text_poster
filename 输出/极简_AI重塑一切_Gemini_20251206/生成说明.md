# 生成说明 (V2 - 极简风格)

## 1. 生成工具与模型
- **工具:** Gemini CLI
- **模型:** Gemini 1.5 Pro

## 2. 生成参数
- **场景:** B站视频封面图
- **尺寸:** 3:4 比例 (宽度 720px, 高度 960px)
- **分辨率:** 通过 `html2canvas` 的 `scale: 2` 参数，生成高分辨率图片。
- **风格:** 极简主义、明亮、专业
- **平台:** Bilibili (B站)
- **文案:** 来源于提供的 `中文海报提示词.md` 文件。

## 3. 整体风格总结
- **设计理念:** 采用明亮、干净、极简的视觉风格，强调内容和可读性。
- **色彩搭配:** 使用米白色背景 (#f8f9fa) 和深灰色文字 (#212529) 创造柔和而清晰的对比。重点关键词和装饰元素采用低饱和度的蓝色 (#0077b6) 作为点缀，整体感觉更为沉稳和专业。
- **字体运用:** 同样使用思源黑体 (Noto Sans SC) 和 Inter 字体，保持品牌一致性和阅读舒适度。
- **排版布局:** 采用左对齐的非对称布局，标题和内容从上至下排列，增加了页面的呼吸感和现代感。章节列表通过左侧的蓝色线条进行视觉引导，使结构更清晰。
- **元素细节:** 舍弃了描边等复杂效果，通过颜色和布局来突出重点，符合极简主义的设计原则。

## 4. 生成思路
1.  **项目结构搭建:** 在 `输出/` 目录下创建独立的文件夹 `极简_AI重塑一切_Gemini_20251206` 以区别于第一个版本。
2.  **HTML 结构:** 复用基本 HTML 结构，但为主要元素添加了 `-v2` 的 class 后缀，以便应用全新的 CSS 样式。
3.  **CSS 样式:** 编写全新的 `style.css`，实现明亮系的配色方案和极简风格的布局。调整了字体大小、边距和对齐方式，以创造不同的视觉感受。
4.  **JavaScript 功能:** 复用 `html2canvas` 逻辑，仅修改了目标元素的 ID (`cover-image-v2`) 和输出文件名 (`bilibili_cover_minimalist.png`)。
5.  **设计迭代:** 作为对第一个暗色主题设计的补充，此版本旨在提供一个风格迥异但同样专业的备选方案。
