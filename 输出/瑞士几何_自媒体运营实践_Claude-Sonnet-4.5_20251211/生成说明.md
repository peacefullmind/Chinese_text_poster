## 3. 整体风格总结

### 设计灵感来源
本设计主要借鉴了两位设计大师的风格特征：

1. **Josef Müller-Brockmann（约瑟夫·米勒-布罗克曼）**
   - 瑞士国际主义平面设计代表人物
   - 特点：严格的网格系统、清晰的层级结构、极简主义
   - 转译手法：使用隐形网格系统组织内容，保持视觉秩序

2. **Piet Mondrian（皮特·蒙德里安）**
   - 荷兰风格派运动代表人物
   - 特点：几何抽象、原色色块、黑色线条
   - 转译手法：运用色块和线条作为装饰元素，不直接复制其画作构图

### 视觉特征
- ✅ **网格系统**: 采用隐形网格控制布局，确保元素对齐
- ✅ **几何形状**: 使用矩形色块和直线营造现代感
- ✅ **色彩方案**: 红色(#E63946)、蓝色(#457B9D)、黄色(#F4A261)、深蓝(#1D3557)、白色
- ✅ **字体层次**: 4 种字号（88px、64px、32px、24px）
- ✅ **视觉焦点**: 通过颜色和字号突出"运营"关键词
- ✅ **留白空间**: 保持适当留白，避免拥挤
- ❌ **无紫色**: 严格遵守要求
- ❌ **无渐变**: 使用纯色色块
- ❌ **无AI图标**: 仅使用几何形状和文字

## 4. 生成思路

### 设计策略

#### 4.1 信息层级
1. **一级信息（最大字号 88px）**: 主标题"自媒体运营实践"
2. **二级信息（32px）**: 关键词标签（运营、实践、方法论）
3. **三级信息（24px）**: 副标题说明
4. **四级信息（14px）**: 顶部英文标签 "PRACTICE"

#### 4.2 视觉引导路径
```
顶部标签 → 主标题（停留） → 关键词强调 → 副标题 → 装饰年份
```

#### 4.3 色彩心理学应用
- **红色**: 热情、行动力，突出"运营"关键词
- **蓝色**: 专业、可信赖，强调实践和方法论
- **黄色**: 创新、活力，作为点缀色增加视觉层次
- **深蓝**: 稳重、正式，主要文字颜色
- **白色**: 简洁、纯粹，背景色

#### 4.4 技术实现要点
1. **响应式设计**: 使用 CSS `aspect-ratio` 严格控制 4:3 比例
2. **字体加载**: Google Fonts CDN 引入思源黑体（Noto Sans SC）
3. **保存功能**: html2canvas 库实现高清截图（3倍分辨率）
4. **交互优化**: 
   - 按钮悬停效果
   - 键盘快捷键（Ctrl/Cmd + S）
   - 保存状态提示
   - 成功/失败通知

#### 4.5 设计转译说明
| 灵感来源 | 原始特征 | 网页实现手法 | 转译理由 |
|---------|---------|------------|---------|
| Müller-Brockmann | 网格系统 | CSS Grid + Flexbox | 保持严谨的视觉秩序 |
| Müller-Brockmann | 无衬线字体 | Noto Sans SC | 现代、清晰、易读 |
| Mondrian | 色块构成 | 绝对定位 div + 透明度 | 营造层次感而非填满画布 |
| Mondrian | 黑色直线 | 半透明线条元素 | 作为装饰而非分割主体 |
| 瑞士风格 | 大胆排版 | 超大字号 + 间距控制 | 确保视觉冲击力 |

## 5. 使用说明

### 5.1 查看效果
1. 直接在浏览器中打开 `index.html` 文件
2. 或使用本地服务器预览

### 5.2 保存图片
- **方法1**: 点击右上角"保存图片"按钮
- **方法2**: 按下键盘快捷键 `Ctrl + S`（Windows）或 `Cmd + S`（Mac）

### 5.3 修改内容
- **修改文案**: 编辑 `index.html` 中的文字内容
- **调整颜色**: 编辑 `styles.css` 中的颜色变量
- **修改布局**: 调整 CSS 中的 `padding`、`margin`、`position` 等属性
