# 生成说明

## 1. 工具与模型
- **生成工具**: Cursor (AI Assisted Programming)
- **调用模型**: Gemini-3-pro-preview

## 2. 生成参数
- **场景**: 抖音合集封面图
- **尺寸**: 4:3 (800px x 600px CSS display, 导出时更高清)
- **分辨率**: 高清 (html2canvas scale: 2)
- **风格**: 瑞士国际主义风格 (Swiss International Style) / 现代极简
- **平台**: 抖音 (Douyin)
- **文案**:
  - 主标题: 自媒体
  - 副标题: 运营实践
  - 装饰文案: MEDIA OPERATIONS / 2025 / PRACTICE & GROWTH

## 3. 整体风格总结
本次设计采用了经典的**瑞士国际主义风格 (Swiss International Style)**，结合了现代数字媒体的视觉需求。
- **色彩策略**: 摒弃了复杂的渐变和紫色（根据要求），采用了极具视觉冲击力的**红 (Red #E63946)**、**黑 (Black)**、**白 (White)** 三色搭配。红色作为主色调能瞬间抓住用户注意力，符合商业营销的高唤醒度要求。
- **排版布局**: 严格遵循**网格系统 (Grid System)**，但引入了轻微的**旋转 (-5deg)**，这是对 Paula Scher 风格的致敬，旨在打破呆板，增加视觉动感，引导视线流动。
- **字体选择**: 选用了无衬线字体 (Inter / Noto Sans SC)，粗重字重 (Black/Bold) 强调力量感和现代感。

## 4. 设计灵感与实现映射 (灵感 -> 网页实现)

| 灵感来源 (Artist/Style) | 设计特征 | 网页代码实现方案 |
| :--- | :--- | :--- |
| **Josef Müller-Brockmann** | 严谨的数学网格，清晰的层级 | `display: grid;` 布局，明确的 `font-size` 阶梯 (9rem, 4.5rem, 1.2rem) |
| **Paula Scher** | 将文字作为图像处理，大胆的错位与旋转 | CSS `transform: rotate(-5deg);`，巨大的 `text-shadow` 制造硬边缘立体感 |
| **Swiss Style (General)** | 几何图形装饰，极简主义 | CSS 伪元素 `::before` 制作背景大字纹理，`div.geometric-line` 绘制装饰线条 |

## 5. 生成思路
1.  **结构搭建**: 确立 4:3 的容器比例，使用 Flexbox 居中显示，便于预览。
2.  **视觉核心**: 提取“自媒体”作为超级符号，使用最大字号占据视觉中心。
3.  **色彩心理**: 红色背景在抖音的信息流中非常醒目，配合黑白文字保证极高的可读性。
4.  **细节打磨**: 添加了 CSS 阴影 (`box-shadow`) 给卡片增加悬浮感（仅预览时），海报内部使用硬阴影 (`text-shadow`) 增加文字质感而不脏。
5.  **功能实现**: 集成 `html2canvas`，通过 JS 脚本实现一键高倍率截图保存，满足“生成图无明显边界”的要求。
