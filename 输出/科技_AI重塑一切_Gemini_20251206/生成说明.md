# 生成说明

## 1. 生成工具与模型
- **工具:** Gemini CLI
- **模型:** Gemini 1.5 Pro

## 2. 生成参数
- **场景:** B站视频封面图
- **尺寸:** 3:4 比例 (宽度 720px, 高度 960px)
- **分辨率:** 通过 `html2canvas` 的 `scale: 2` 参数，生成高分辨率图片。
- **风格:** 科技感，现代，注重视觉冲击力和可读性。
- **平台:** Bilibili (B站)
- **文案:** 来源于提供的 `中文海报提示词.md` 文件。

## 3. 整体风格总结
- **设计理念:** 采用简洁、现代、科技感的视觉风格。
- **色彩搭配:** 以深色背景 (#1a1a1a) 和浅色文字 (#f0f0f0) 形成强烈对比，辅以醒目的橙色 (#ff5722) 作为关键词的强调色，避免使用紫色和渐变色。
- **字体运用:** 中文标题使用思源黑体 (Noto Sans SC)，副标题和章节列表使用 Inter 字体，确保文字的清晰度和现代感。
- **排版布局:** 采用垂直居中布局，通过不同字号和字重创建清晰的文本层级，引导观众视线。
- **元素细节:** 关键词通过描边和颜色突出，增加视觉焦点。整体设计保持卡片边缘锐利，符合简洁专业的要求。

## 4. 生成思路
1.  **项目结构搭建:** 在 `输出/` 目录下创建独立的文件夹 `科技_AI重塑一切_Gemini_20251206`。
2.  **HTML 结构:** 构建 `index.html`，包含封面图的主体容器 (`#cover-image`)、主标题、章节列表及一个保存按钮。引入 Google Fonts 和 Font Awesome 的 CDN 链接，并预留 `style.css` 和 `script.js` 的引用。
3.  **CSS 样式:** 编写 `style.css`，实现 3:4 的宽高比、字体样式、颜色方案、文本排版和基础布局。特别处理了关键词的样式，使其在视觉上更突出。
4.  **JavaScript 功能:** 创建 `script.js`，利用 `html2canvas` 库实现将 `#cover-image` 区域渲染为图片并提供下载的功能。设置 `scale: 2` 以提高输出图片的分辨率。
5.  **关键词处理:** 在 `index.html` 中，将主标题的关键词用 `<span>` 标签包裹，以便 CSS 可以对其进行独立样式设置。
6.  **合规性检查:** 确保设计严格遵循了“禁止使用紫色、渐变色”和“3:4 宽高比”等所有需求。
